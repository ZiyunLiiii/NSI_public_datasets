NSI 4D Phantom CBCT: NSI Phantom Dynamic Scan

Dynamic cone-beam CT scan of a 4D phantom for evaluating time-resolved CT reconstruction methods.

Data:

A high-frame-rate cone-beam CT acquisition of a dynamic phantom collected on an NSI PRD-K416090 system over multiple rotations.

NSI 4D CT Phantom - 2400 views

Preprocessing:

Example preprocessing and reconstruction scripts are available in the nsi folder of the MBIRJAX applications repository.

These scripts reconstruct the NSI datasets using the MBIRJAX software package available from MBIRJAX Documentation.
